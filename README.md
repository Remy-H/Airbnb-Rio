# Airbnb Rio - Prédiction des Prix

**Projet Machine Learning** : Prédire la gamme de prix des logements Airbnb à Rio de Janeiro (35 602 listings).  

Développer un modèle qui classe les logements en 4 gammes de prix selon leurs caractéristiques.  

## Technologies utilisées
  Python + Pandas + NumPy (data manipulation)  
  Scikit-learn (GridSearchCV 100 modèles)  
  SHAP (interprétabilité : force plots, importance variables)  
  Matplotlib/Seaborn (EDA + visualisations)
